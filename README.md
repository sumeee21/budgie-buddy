# 💸 Budgie Buddy

Budgie Buddy is a **voice-first AI-powered personal finance companion** designed for college students. It helps users track expenses through natural conversation, manage monthly budgets, enforce daily limits, and build better financial habits.

---

## 🚀 Features

- 🎤 Voice + Text Expense Logging  
- 🧠 AI-based Expense Understanding (Natural Language Processing)  
- 💰 Real-time Budget Tracking  
- 📊 Automatic Expense Categorization (Food, Transport, Education, etc.)  
- ⚠️ Daily Spending Limit Alerts  
- 🎉 Smart Feedback & Motivation  
- 🔔 Notifications (No-spend day, overspending alerts)  
- ☁️ Cloud Data Storage  

---

## 🧠 How It Works

1. User sets monthly budget (e.g., ₹5000)  
2. User logs expenses using voice or text  
3. AI processes input and extracts amount + category  
4. Expense is saved and categorized  
5. Remaining balance updates instantly  
6. AI provides feedback (encouragement or warning)  
7. Notifications are sent based on user activity  

---

## 🏗️ Tech Stack

- **Frontend:** Next.js  
- **Backend:** Firebase / Node.js  
- **Database:** Firebase Firestore  
- **Styling:** Tailwind CSS  
- **AI/NLP:** OpenAI API  
- **Voice Input:** Speech-to-Text API
